Case Study

Problem

Small businesses need an affordable, full-featured e-commerce platform without monthly subscription fees.

Solution

Developed a complete e-commerce platform with Angular frontend and Node.js backend with MongoDB featuring product management, shopping cart, order tracking, and admin dashboard.

Result

Self-hosted e-commerce platform with JWT authentication, role-based access, and full CRUD product management.

Features

JWT Authentication

Secure authentication with JWT token-based authorization.

Product Catalog with Categories

Full CRUD for products with category filtering and search.

Shopping Cart

Persistent cart with quantity controls and total calculation.

Order History

Order tracking with status updates.

Admin Dashboard

Order management dashboard for administrators.

Role-based Auth

Role-based access with Admin and Customer roles via Identity policies.

Tech Stack

Angular 18 TypeScript MongoDB JWT

Quick Start

Terminalbash
# Clone the repository
git clone https://github.com/mostafaali/ecommerce-platform.git
cd ecommerce-platform

# Set up the backend
cd backend
dotnet restore
dotnet run
# Database is auto-created via EnsureCreated()

# In another terminal – set up the frontend
cd frontend
npm install
ng serve
