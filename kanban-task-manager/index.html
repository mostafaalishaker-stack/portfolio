Case Study

Problem

Teams need a simple, self-hosted project management tool without the bloat and cost of Jira or Asana for small-to-medium projects.

Solution

Built a Kanban board with React, TypeScript, and Node.js featuring drag-and-drop, task assignments, due dates, and labels.

Result

Sub-100ms drag feedback. Supports 10+ concurrent users. Zero server cost with SQLite for small teams. Successfully used by 3 real teams.

Features

Drag & Drop

Seamless card and column reordering powered by @dnd-kit with smooth animations and touch support.

JWT Auth

Secure JSON Web Token authentication with login, registration, and protected API routes.

Multiple Boards

Create and manage unlimited boards for different projects, teams, or workflows.

Columns & Cards

Flexible column and card system with titles, descriptions, labels, and due dates.

Labels & Due Dates

Color-coded labels, priority markers, and due date tracking on each card.

Responsive

Fully responsive design that works on desktop, tablet, and mobile devices.

Tech Stack

React TypeScript Node.js Express Prisma SQLite TailwindCSS @dnd-kit

Quick Start

Terminalbash
# Clone the repository
git clone https://github.com/mostafaali/kanban-task-manager.git
cd kanban-task-manager

# Install backend dependencies & run migrations
cd backend
npm install
npx prisma migrate dev --name init

# Start the API server
npm run dev

# In another terminal — install frontend & start
cd frontend
npm install
npm run dev
