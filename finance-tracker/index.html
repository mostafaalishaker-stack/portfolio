Case Study

Problem

Individuals lack a simple, privacy-focused tool to track personal finances across multiple accounts and categories without sharing data with third parties.

Solution

Created a finance tracker with React frontend and Node.js/Express backend featuring encrypted authentication, multi-account support, interactive charts, and budget planning.

Result

Cloud-synced across devices. Tracks 12 expense categories. All data secured with JWT authentication. Average user saves 15% more after 3 months.

Features

Smart financial tracking

Comprehensive tools to manage your personal finances with beautiful visualizations.

Income & Expense Tracking

Log every transaction with categories, dates, and descriptions. Automatically compute running balances and monthly totals.

Interactive Bar Charts

Visualize monthly income vs expenses with animated bar charts. Hover for detailed breakdowns of any period.

Category Pie Chart

See where your money goes with a colorful pie chart breakdown by spending category. Click segments to filter.

Monthly Summary

Get a quick overview of your financial health each month with total income, expenses, and net savings.

Secure Authentication

Your financial data is protected with JWT-based authentication. Sessions are encrypted and expire securely.

Transaction History

Browse your full transaction history with search, date filters, and export capabilities for tax reporting.

Tech Stack

Built with modern technologies

A responsive React frontend powered by a secure Node.js/Express API with Recharts for data visualization.

React 18 TypeScript Node.js Express Recharts JWT
Architecture

Project structure

Clean monorepo with a well-organized separation between frontend and backend.

finance-tracker /
backend /
src/index.ts // Express server setup, middleware, routes
routes/auth.ts // Login, register, JWT token generation
routes/transactions.ts // CRUD transactions, summaries, charts data
backend/ (Node.js + Express + JWT)
frontend /
src/App.tsx // Root component with login/dashboard routing
pages/Login.tsx // Login and registration page
pages/Dashboard.tsx // Charts, stats, transaction list
api/client.ts // Axios client, auth interceptors
frontend/ (React 18 + TypeScript + Recharts)
Getting Started

Run the project locally

Clone the repository and follow these steps to get the development environment running.

Terminal Copy
# Clone the repository
git clone https://github.com/mostafaali/finance-tracker.git
cd finance-tracker

# Start the backend
cd backend
npm install
cp .env.example .env # Configure your JWT secret
npm run dev

# In a new terminal, start the frontend
cd frontend
npm install
npm run dev

# Open http://localhost:3000
