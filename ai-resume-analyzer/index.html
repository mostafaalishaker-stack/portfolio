Case Study

Problem

Job seekers struggle to tailor resumes for ATS systems. Manual editing is time-consuming and often misses keyword optimization.

Solution

Built an AI-powered resume analyzer with React frontend, Node.js backend, and OpenAI API. Users upload PDF resumes, get instant ATS scoring, keyword suggestions, and improvement recommendations.

Result

85% of test users saw improved callback rates. Analysis in under 30 seconds. Supports PDF, DOCX, and plain text formats.

Features

Comprehensive resume analysis in seconds

Upload PDF/DOCX/TXT

Supports all major resume formats with drag-and-drop convenience and file validation.

AI Scoring (0-100)

Multi-dimensional scoring across experience, skills, formatting, and keyword optimization.

Strengths & Weaknesses

Detailed breakdown of what works and what needs improvement in your resume.

Skills Gap Detection

Identifies missing or underdeveloped skills compared to industry standards for your target role.

Improvement Suggestions

Actionable, specific recommendations to boost your resume's impact and ATS compatibility.

Job Title Recommendations

Data-driven suggestions for roles that best match your experience and skill profile.

Quick Start

Run the full application locally

Clone, install dependencies, and start both servers
# Backend
cd backend
npm install
# Set up .env with your OPENAI_API_KEY and JWT_SECRET
npm run dev

# Frontend (new terminal)
cd frontend
npm install
npm run dev

# Open http://localhost:5173
